A Little Background

I'm Amed — a Software Engineering graduate from Bahcesehir University and a cybersecurity professional based in Istanbul. I've spent the last few years building hands-on experience across SOC analysis, Application Security, SIEM management, DLP administration, and OSINT-based penetration testing.

During my time at Halkbank, I worked long-term within the System Security, SOC, and Cybersecurity teams — contributing to alert triage, DLP monitoring, static code analysis, and black-box security assessments. Earlier at TFS Group, I managed the entire internal network infrastructure independently.

I'm interested in roles across Network Security, AppSec, SOC Analysis, Penetration Testing, GRC, and AI Security. When I'm not deep in a terminal, I'm producing music, attending security conferences across Europe, or contributing to the local hacker community.
